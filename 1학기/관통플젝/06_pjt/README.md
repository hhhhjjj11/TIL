# 새로 배운 것들 및 느낀점
## 리마인드
  - 로그인시
    AuthenticationForm
    - 인자 두개
    - (req, req.POST)

  - aut_login
    - 인자두개
    - (req, form.get_user())

  - 회원가입시
    - 유저 인스턴스를 리턴받아서 로그인함수에 인자로 넣는다는 점
    ```
    user = form.save()
    auth_logn(req, user)
    ```

  - 비밀번호 변경시
    - 수정할때와 비슷하게, 빈폼에도 req.user인자로 넣어줘야 한다는 점
    ```
        form = PasswordChangeForm(req.user)
    ```

  - 메소드옵션 포스트로 설정한 폼태그 안에 a태그넣으면 포스트요청간다..
